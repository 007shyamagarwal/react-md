    Make a material design default button This will generate the style for a flat, raised, and fab button

    Parameters

    $color
      String Required

    The text color to use

    $hover-focus-bg-color
      String Required

    The hover and focus color for the button

    $active-bg-color
      String Required

    The color to use when active

    $md-ink-bg-color
      String Required

    The md-ink background color

    $raised-border-color
      String Default:null

    The raised border color. If this value is set, raised and floating buttons will get a background of the white base and the raised buttonm with get a border with the border color

    md-theme-buttons 

    @mixin md-theme-buttons($color, $text-color: $white-base, $hover-color: rgba($color, .95), $ink-color: rgba($white-base, .05), $flat-hover-color: rgba($color, .15), $flat-ink-color: rgba($color, .22)) { ... }

    Themes the material design buttons with the given color

    Parameters

    $color
      Color Required

    The background color to use for raised and floating buttons and the text color to use for flat buttons.

    $text-color
      Color Default:$white-base

    The text color for raised and floating buttons.

    $hover-color
      Color Default:rgba($color, .95)

    The hover color for raised and floating buttons. If this value is set to null, it will use the default listed.

    $ink-color
      Color Default:rgba($white-base, .05)

    The ink color for raised and floating buttons. If this value is set to null, it will use the default listed.

    $flat-hover-color
      Color Default:rgba($color, .15)

    The hover color for flat buttons. If this value is set to null, it will use the default listed.

    $flat-ink-color
      Color Default:rgba($color, .22)

    The ink color for flat buttons. If this value is set to null, it will use the default listed.

    Example

    Example Basic Usage SASS
    

    .md-primary {
  @include md-theme-buttons(get-md-color(indigo));
}
    Example Basic Usage CSS Output
    

    .md-primary.md-raised-btn,
.md-primary.md-floating-btn {

  color: #fff;

    get-md-color 

    @function get-md-color($color, $hue: 6, $accent: false) { ... }

    Get a material design color for the color palette. This will be able to find a primary or an accent color for all the colors on the material design color palette page.

    The only exceptions are brown, grey, and blue-grey. These will only return their primary colors. Black and white will work as an rgba instead on their base colors where $hue is the opacity level.

    Parameters

    $color
      String Required

    One of the material design colors to look up. Should be one of (red pink purple deep-purple indigo blue light-blue cyan teal green light-green lime yellow amber orange deep-orange brown grey blue-grey).

    $hue
      Number Default:6

    This is the color mapping from the material design color palette page. This value should be in the range from 1 - 10 when this is a primary color. 1 would return the color with a strength of 50, while 2 - 10 would return a color with a strength of a multiple of 100. The value should be in the range from 1 - 4 when it is an accent color. Finally, the value should be a number from 0 - 1 when the color is black.

    $accent
      Bool Default:false

    Is this color an accent color?

    Returns

    the material design color code

    String

    Example

    Basic Usage Sass
    

    .some-primary-bg {
  background: get-md-color(indigo);
}
    Basic Usage CSS Output
    

    .some-primary-bg {
  background: #3F51B5;
}
    Primary Color with Specific Hue Sass
    

    .some-bg {
  background: get-md-color(deep-purple, 3);
}
    Primary Color with Specific Hue CSS Output
    

    .some-bg {
  background: #9fa8da;
}
    Accent Color Sass
    

    .some-bg {
  background: get-md-color(teal, 2, true);
}
    Accent Color CSS Output
    

    .some-bg {
  background: #64ffda;
}
    Black color Sass

  background: get-md-color(black, .26);
}
    Black Color CSS Output
